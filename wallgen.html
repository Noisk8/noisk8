Generative Image con wallgen | python

Genera fondos de pantalla

Crear fondos de pantalla de poliéster sobre un degradado con colores aleatorios o utilizando colores personalizados.

Elige diferentes formas como hexágonos, cuadrados y diamantes aparte de los triángulos.

Aplicar polígonos sobre una imagen

Para Instalar en linux

Requerimientos

via pip

pip install -e git+https://github.com/SubhrajitPrusty/wallgen#egg=wallgen

Clona el repositorio

git clone https://github.com/SubhrajitPrusty/wallgen.git

cd wallgen

pip install --editable

Uso

Usage: wallgen [OPTIONS] COMMAND [ARGS]...

Opciones:

-h, --help Mostrar este mensaje y salir.

pic Use una imagen en lugar de un degradado

poly Genera una imagen HQ low poly usando un gradiente

shape Genera una imagen de HQ de unas hermosas formas

slants Genera líneas oblicuas de varios colores

Ejemplos

Degradado aleatorio

wallgen poly 2000

Método NbyNDegradado

wallgen poly 2000 --use-nn --points 5000

Color fijo/degradado

wallgen poly 1000 --colors "#ff0000" --colors "#00ddff"

Remolino

wallgen poly 2000 -c "#ff0000" -c "#00ddff" --points 2000 --swirl

En vídeo

Wall gen tambien se puede usar para hacer poly en video, pasando este script

python3 nombredelscript.py nombre delarchivo.mp4

El resultado son tres archivos un .avi con audio y video, un .mp3 que solo contiene el audio y un .pm4 que contiene solo el video
